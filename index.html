Arquitectura y Rendimiento

✂️ "UI Tijeretazo" (Seguridad Frontend)

La interfaz se adapta dinámicamente según el rol (RBAC). Aislamos al técnico de funciones administrativas, ocultando botones destructivos para reducir la carga cognitiva.

🪶 Iconografía Ligera (Unicode)

Prescindimos de librerías pesadas como FontAwesome. Usamos Emojis/Unicode nativos del sistema (⚙️, 🔧, 🖨️) para lograr un First Contentful Paint (FCP) casi instantáneo.

📄 PDF Renderizado en Cliente

Liberamos al servidor. El documento PDF se construye en el propio navegador del dispositivo (Canvas API + Blob Objects), ahorrando costes masivos de procesamiento en la nube.
